# tasks-mvp

Simple and easy to use console based task manager.

## Installation

To build

```bash
make
```

then 

```bash
make install
```

## Usage

Add simple todo item
```bash
tsk add -m "Listen to new  single from Weki Meki"
```

Add cron based task

```
tks add -m "Wacth new episode of Black Pink house" -c "0 0 * * MON"  
```
